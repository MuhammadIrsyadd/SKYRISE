# SKYRISE 🏗️🏙️

**"Build Your Legacy, One Floor at a Time"**

SKYRISE is a high-octane, arcade-style tower stacking game built with modern web technologies. Challenge your reflexes and timing as you construct the tallest skyscraper the city has ever seen. Each floor is a unique architectural piece, and one wrong move could bring your legacy crumbling down!

## 🎮 How to Play

1.  **Start the Game:** Click the "START BUILDING" button on the home screen.
2.  **Drop Blocks:** A block will move horizontally across the screen. **Click**, **Tap**, or press the **Spacebar** to drop it onto the tower.
3.  **Align Precisely:** Try to land the block perfectly on top of the previous floor.
    *   **Perfect Hit:** If you align it perfectly, you get a score bonus and a gold particle burst!
    *   **Off-Center:** Any part of the block that hangs over the edge will be trimmed off, making your tower narrower.
4.  **Difficulty Scaling:** As you go higher, the blocks will move faster, testing your precision to the limit!
5.  **Game Over:** The game ends if you miss a block entirely or if the tower becomes too narrow to support another floor.

## ✨ Features

-   🌈 **Vibrant Variety:** Every single floor features a random architectural style and color palette (12+ styles available).
-   ⚡ **Progressive Difficulty:** Dynamic speed scaling that increases as you climb higher.
-   ☁️ **Atmospheric Sky:** A beautiful day-night cycle that changes based on your tower's height (Dawn to Midnight).
-   🎨 **Smooth Visuals:** High-performance HTML5 Canvas rendering with a 60FPS game loop.
-   🔉 **Procedural Audio:** Dynamic sound effects generated via the Web Audio API.
-   🏆 **High Score Tracking:** Save your best runs locally and try to beat your own record.
-   📱 **Fully Responsive:** Playable on both Desktop and Mobile devices.

## 🛠️ Technical Stack

-   **Language:** Vanilla JavaScript (ES2022+)
-   **Rendering:** HTML5 Canvas 2D API
-   **Build Tool:** [Vite](https://vitejs.dev/)
-   **Icons:** [Tabler Icons](https://tabler-icons.io/)
-   **Styling:** Modern CSS3 with Flexbox and Grid

## 🚀 Local Development

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/MuhammadIrsyadd/SKYRISE.git
    ```
2.  Navigate to the directory:
    ```bash
    cd SKYRISE
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
